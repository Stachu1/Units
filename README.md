# Units

Clone:
```bash
git clone https://github.com/Stachu1/Units.git
```

Run:
```bash
python units.py "N*m*s^-1"
```

![image](https://github.com/user-attachments/assets/96f25a9f-deda-48e3-8939-d38e6ff3474c)


Supported units:
```
N - Newton
J - Joule
W - Watt
C - Coulomb
V - Volt
ohm - Ohm
F - Farad
H - Henry
T - Tesla
Wb - Weber
Hz - Hertz
Pa - Pascal
lx - Lux
S - Siemens
Gy - Gray
Sv - Sievert
kat - Katal
lm - Lumen
sr - Steradian
m - Meter
s - Second
kg - Kilogram
A - Ampere
K - Kelvin
mol - Mole
cd - Candela
```
